# Kerberos
[![Build Status](https://travis-ci.org/Aethylred/puppet-kerberos.svg?branch=master)](https://travis-ci.org/Aethylred/puppet-kerberos)

This Puppet module manages and installs MIT Kerberos 5 authentication.
